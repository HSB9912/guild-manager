변경 포인트

  • 맨 위 새 박스: 수로 상시 이벤트 안내 (4줄 핵심) + 슬라이더
  • 표기: K → 만점 (1만점 / 2.2만점 / 3.8만점 …)
  • 마커: 1만점(200등·진입) / 1.3만(180등·강등 경계) / 3.8만(90등·안정선) / 4.5만 평균 / 6.5만 티라미슈 / 9.5만 크라운
  • 구간 정의: 강등권 180~200등 / 변동성 90~180등 / 안정권 90등 이내
  • 슬라이더 배경 색구간: 회색(불가) → amber(강등) → yellow(변동) → 연두(안정) → 핑크(상위)
  • 슬라이더 범위: 0~200,000 · 100K까지 1만 단위, 100K~200K는 50K 단위 (사용자 요구)
  • 예상 위치: 슬라이더 값에 따라 자동 — 가입 불가 / 강등 위험 / 안정권 / 중상위 / 상위 10% / 크라운권
  • 기존 폼·핸즈·질문·12시간 대기·10K 가드 전부 살림
  • 실제 적용 시 members.suro 분기 평균에서 160등·평균·TOP5 등 동적 계산 (현재 미리보기는 임시값)
