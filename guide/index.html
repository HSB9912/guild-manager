길드 신청 시스템 사용 안내

뚠 / 뚱카롱 길드 · 면제 신청 + 보석금 납부 매뉴얼

사이트: hsb9912.github.io/guild-manager

언제 무엇을 써야 하나요
아인슈페너 신청

이번 주 수로 점수가 부족할 것 같을 때
→ 부캐 직위를 아인슈페너로 전환해 노블 사용 가능
보석금 납부

이미 미참해서 노블이 막혔을 때
→ 길드창고에 솔 에르다 조각 납부 후 인증

두 신청 모두 운영진이 검토 후 처리합니다. 진행 상태는 페이지 하단 "내 신청 내역"에서 실시간 확인 가능합니다.

1

수로 면제 신청 — 아인슈페너

본인 본캐 닉 하나만 입력하면 점수 자동 조회 → 부캐 자동 분류까지 처리됩니다. 운영진이 승인하면 부캐 직위가 아인슈페너로 자동 변경되어 노블을 쓸 수 있습니다.

페이지 주소
hsb9912.github.io/guild-manager/exempt/
1
페이지 열기
위 주소를 입력하거나 메인 사이트의 "면제 신청" 메뉴를 선택합니다.
2
본캐 닉 입력 → 돋보기 버튼
본인의 본캐(메인 캐릭) 닉네임만 입력하면 됩니다. 부캐 닉은 자동으로 가져옵니다.
3
자동 분류 결과 확인
본캐 점수에 따라 신청 타입이 결정됩니다.
  • • 9만점 이상 — 전체 면제 (모든 부캐 자동 적용)
  • • 9만점 미만 — 절반 면제 (면제할 부캐 직접 선택)
4
사유 / 오픈챗 닉 입력 (선택)
필수 아닙니다. 알릴 점이 있을 때만 작성하시면 됩니다.
5
신청하기 버튼 클릭
완료. 운영진이 검토 후 승인하면 직위가 자동 변경됩니다. 처리 결과는 페이지 하단 "내 신청 내역"에 실시간 표시됩니다.
2

보석금 납부 신청 — 노블 해제

수로 미참 시 노블이 잠깁니다. ★보석금★ 길드 창고에 솔 에르다 조각을 납부하고 인증 캡처를 첨부하면, 운영진이 게임 안에서 노블을 해제해 드립니다.

진행 순서
1

길드창고에 입금

  1. 게임 접속 후 ★보석금★ 길드 창고 NPC로 이동
  2. NPC 대화 → 길드 창고 열기
  3. 본인 인벤토리(기타 탭)에서 솔 에르다 조각을 끌어다 길드창고에 입고
2

입금 인증 캡처

길드창고 좌측의 GUILD STORAGE LOG 패널에 다음과 같은 줄이 본인 닉네임으로 표시되어야 합니다.

[날짜 시간] OOO님 님이 솔 에르다 조각 (N개) 을 입고하였습니다

이 줄이 보이는 상태에서 화면을 캡처해 주세요.

인정되는 캡처
  • GUILD STORAGE LOG에 본인 닉 표시됨
  • 솔 에르다 조각 수량 보임
  • 입고 시간 보임
인정되지 않음
  • 메소 송금 화면
  • 본인 인벤토리만 캡처
  • 채팅창 캡처
  • 닉네임 잘려서 안 보임
3

신청 페이지에서 제출

  1. 보석금 페이지 열기 (위 주소)
  2. 본캐 닉 입력 → 돋보기 버튼
  3. 미참한 캐릭 체크 (여러 캐릭 동시 가능 — 합계와 누진세 자동 계산)
  4. 입금 인증 캡처 첨부
    • PC: 캡처 후 Ctrl + V 한 번이면 자동 첨부
    • 모바일: 첨부 영역을 눌러 갤러리/카메라 선택
  5. (선택) 사유 / 오픈챗 닉 입력
  6. "보석금 납부 신청" 버튼 클릭
처리 단계 (페이지에서 실시간 확인)
확인 대기 입금 확인됨 노블 해제 완료

자주 묻는 질문

본캐 검색해도 안 나옵니다
사이트에 캐릭이 아직 등록되지 않은 경우입니다. 운영진에게 등록을 먼저 요청해 주세요. 면제 신청 페이지에서는 신규 가입자도 함께 신청 가능한 옵션이 있습니다.
부캐 여러 개가 동시에 미참했어요. 따로 신청하나요?
한 번에 처리됩니다. 보석금 페이지에서 미참한 부캐를 모두 체크하면 합계가 자동 계산됩니다. 길드창고에 합계만큼 한 번에 입금하고, 캡처 한 장만 첨부하면 됩니다.
신청 후 얼마나 기다려야 하나요?
운영진이 확인하는 대로 처리됩니다. 페이지 하단 "내 신청 내역"에서 진행 상태(확인 대기 → 입금 확인됨 → 노블 해제 완료)를 실시간으로 보실 수 있습니다.
누진세가 잘못 계산된 것 같아요
페이지에 표시되는 횟수는 같은 반기에 신청한 이력 기준입니다. 사정이 있으시면 사유 칸에 적어주시거나 운영진에게 문의해 주세요.
잘못 신청했습니다. 취소되나요?
운영진이 처리 전이라면 취소 가능합니다. 카톡 또는 디스코드로 운영진에게 알려 주세요.
